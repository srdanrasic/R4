R4View Class Reference

Inherits from UIView
Declared in R4View.h

Overview

An R4View object is a view that displays 3D scene content. This content is provided by an R4Scene object.

You present a scene by calling the view’s presentScene: method. When a scene is presented by the view, it alternates between running its simulation (which animates the content) and rendering the content for display. You can pause the scene by setting the view’s paused property to YES.

For the non-documented items refer to the SKView class reference.

Parameters

Return Value

convertPoint:toScene:

Converts a point from view space (bounds) into a ray in scene space.

- (R4Ray)convertPoint:(CGPoint)point toScene:(R4Scene *)scene

Parameters

point

Point in view space.

scene

A scene.

Return Value

A ray in scene space.

Discussion

A point on 2D projection of the scene corresponds to a ray in scene space that intersects camera’s near and far clip planes.
